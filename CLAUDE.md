## 项目概述

**web_sip** — Web Self Improvement Platform（Web 端自我提升平台）

一个跨平台的自我管理工具，最终目标是在 Web 端运行，让用户可以随时查看、更新和记录自己的计划制定与履行情况。核心理念：制定计划 → 执行跟踪 → 回顾验收 → 持续改进。

## 技术愿景

- **目标平台**: Web 端（跨平台使用，PC + 移动端）
- **核心功能**: 计划制定、任务跟踪、成果验收、进度可视化
- **当前状态**: 项目初始化阶段，技术栈待定

## 工作规矩

1. **语言**: 尽量使用中文交流
2. **Python 命令**: 执行 Python 脚本时使用 `python` 命令，不要使用 `python3`（本机为 Python 3 环境，但 `python3` 命令不可用）
3. **先规划后执行**: 开展任何新特性或功能前，必须先做规划并提交确认，得到确认后再执行
