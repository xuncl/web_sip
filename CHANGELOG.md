# Changelog

## v1.0 (2026-06-12)

### 🎉 首次发布

**SIP 得分系统**
- 连击递增机制：完成任务次日分值增加，越坚持收益越高
- 中断惩罚：关键任务未完成扣分并重置，非关键任务豁免
- 累计总分机制，下限为 0

**任务管理**
- 9 条初始任务模板（工作/事业/自我提升/家庭生活/主动性/锻炼节食/早起/日摘/思考冥想）
- 关键/非关键任务区分
- 完成状态切换，实时计算总分
- 备注字段

**数据持久化**
- GitHub API 读写 JSON，以日期命名
- 自动推导新一天的任务和分值
- 本地缓存兜底

**严格模式**
- 中断天数逐日模拟扣分
- 1~30 天自动补录 JSON 文件
- >30 天总分归零重启

**日期导航**
- ◀ ▶ 按钮浏览任意日期
- 修改历史后自动级联更新到今天

**技术**
- 纯静态 HTML + 原生 JS，零依赖
- GitHub Pages 部署
- 移动端优先响应式布局
- 4 种任务卡片视觉状态
